Syntax:
Exp u ::= x                         variable
          λx.u                      abstraction
          u1 u2                     application
          true                      boolean
          false                     boolean
          if                        stuff
          [0-9]+                    number
          add1                      stuff
          sub1                      stuff
          zero?                     stuff

          maybe other stuff         also stuff
e.g.

(λa.λb.λc.a c c) foo bar quux

add1 3

zero? 0

if true foo bar

(λx.(if (zero? (sub1 x)) add1 sub1)) 1 5
